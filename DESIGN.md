# DESIGN.md

## Project Direction

This project is a visual-first portfolio experience focused on premium presentation, typography, spacing, and emotional clarity.

The design language should feel:
- minimal
- editorial
- premium
- spacious
- intentional
- modern but timeless

The visual tone should resemble:
- Squarespace
- modern editorial portfolios
- high-end creative agency websites
- premium SaaS landing pages

Avoid startup-style clutter and generic AI-generated layouts.

---

# Core Principles

## 1. Typography First

Typography is the primary visual element.

Large headlines should dominate the layout and create visual identity before imagery or decoration.

Use:
- oversized headings
- tight line-height
- elegant serif accents
- strong hierarchy
- intentional whitespace

Typography should feel designed, not merely readable.

---

## 2. Restraint Over Decoration

Do not rely on:
- excessive gradients
- glowing effects
- glassmorphism
- heavy shadows
- colorful UI
- floating random shapes

The interface should feel calm and controlled.

Every visual element must have purpose.

---

## 3. Premium Spacing

Whitespace is a major design component.

Use:
- large section paddings
- generous vertical rhythm
- breathing room around typography
- balanced layout density

Avoid cramped UI.

---

## 4. Editorial Layout

Layouts should feel asymmetrical but balanced.

Prefer:
- split layouts
- offset imagery
- intentional alignment systems
- visual tension between elements

Avoid generic centered SaaS hero structures.

---

# Visual Identity

## Color System

The palette should remain restrained and neutral.

Primary colors:
- warm white
- soft off-white
- charcoal black
- muted gray
- subtle beige tones

Accent colors should be minimal and intentional.

Do not introduce multiple competing accent colors.

### Design Goals
- soft contrast
- elegant readability
- premium atmosphere
- low visual noise

---

# Typography System

## Heading Style

Headings should feel:
- bold
- elegant
- cinematic
- confident

Use a combination of:
- modern sans-serif
- refined serif italic accents

### Preferred Treatment
- large display text
- mixed font pairing
- occasional italic emphasis
- high contrast sizing

Example behavior:
- main heading in bold sans-serif
- one emotional keyword in serif italic

---

## Body Text

Body copy should:
- remain calm and secondary
- use muted foreground colors
- prioritize readability
- never compete with headlines

Avoid overly small text.

---

# Layout Direction

## Hero Section Structure

Preferred structure:
- left content column
- right visual column
- large portrait or artistic image
- floating micro-card or status element
- top minimal navigation
- subtle trust indicators/logos below hero

### Hero Goals
The hero should immediately communicate:
- professionalism
- confidence
- premium craft
- calmness
- trust

The user should feel:
"This person builds refined digital experiences."

---

# Components

## Buttons

Buttons should feel:
- compact
- premium
- tactile
- minimal

Primary button:
- dark background
- light foreground
- medium radius
- subtle hover transition

Secondary button:
- low emphasis
- text-first
- understated interaction

Avoid oversized colorful buttons.

---

## Cards

Cards should:
- use minimal shadows
- rely more on borders and spacing
- feel lightweight
- preserve layout cleanliness

Avoid deeply nested cards.

---

## Navigation

Navigation should:
- remain minimal
- contain few items
- use subtle typography
- prioritize whitespace

The navbar should never dominate the hero.

---

# Motion & Interaction

Motion should be:
- subtle
- smooth
- premium
- low-energy

Preferred motion:
- fade-in
- soft upward movement
- gentle scale transitions
- smooth hover opacity

Avoid:
- bouncy animations
- exaggerated transforms
- fast motion
- distracting effects

---

# Imagery Direction

Imagery should feel:
- editorial
- cinematic
- calm
- premium

Preferred imagery:
- portrait photography
- soft natural lighting
- neutral tones
- clean backgrounds
- fashion/editorial framing

Avoid:
- stock-looking business imagery
- oversaturated colors
- futuristic neon visuals

---

# Technical Direction

Build UI using:
- React
- TailwindCSS
- Framer Motion

Prefer:
- reusable sections
- clean component structure
- semantic spacing system
- responsive layouts

Do not over-engineer.

---

# Anti-Patterns

Avoid:
- generic AI SaaS layouts
- excessive gradients
- too many cards
- too many colors
- giant glowing borders
- random floating elements
- cluttered dashboards
- excessive blur effects
- weak typography hierarchy
- cramped spacing
- overuse of centered layouts

---

# Desired Emotional Outcome

The final UI should feel:
- calm
- expensive
- modern
- intentional
- trustworthy
- editorial
- refined

The interface should communicate craftsmanship rather than trend-chasing.
